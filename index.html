Mission

市民・企業・行政の三者による、地域課題の解決。

Code for Toyama City は、私たちひとりひとりが主体的に富山の地域課題を捉え、手を動かしながら共に解決していくことを目指します。

「富山を変えたい。」と思っても、一人の力では弱いかもしれません。しかし、富山に関わる市民・企業・行政などあらゆる人々が、それぞれの立場を超えて協力しあうと社会は必ず変わっていきます。

私たちと同じく、ITで地域課題の解決に挑戦する地域版の Code for が日本全国にあり、世界各地にも各国版の Code for があります。Code for Toyama City は、そんな仲間たちとも連携しながらワクワクする未来を創るための環境を提供してきました。

共により良い未来を創っていきませんか？活動にご興味のある方は、お気軽にcodefortoyama@gmail.comまでご連絡ください。
